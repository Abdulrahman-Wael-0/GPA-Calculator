# 🎓 GPA Calculator (C++) 🎓

---

## 📝 Project Description

The **GPA Calculator** is a powerful tool designed to help students **compute their Grade Point Average** based on the grades they receive in different courses and the credit hours of each course.  

This program is written in **C++** and uses the **windows.h** library to display **colorful text** and a **clean, tabular interface** in the console.  

It demonstrates:  
- **Object-Oriented Programming**  
- **User Input Handling**  
- **Data Formatting and GPA Calculations**

---

## ⚙ Features

- Enter multiple courses with grades and credit hours  
- Calculate **Semester GPA** or **Cumulative GPA (CGPA)**  
- Display results in a **neatly formatted colored table**  
- Learn and explore **OOP concepts** in C++
